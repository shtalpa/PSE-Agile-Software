# PSE-Agile-Software

### Project title: Where's My Water?
### SDG Goal: 
SDG 6 -- Software can monitor water quality and manage community resources.

### Project Description: 
This application helps university students locate the nearest clean water stations and allows them to report maintenance issues. By ensuring water stations are functional, we reduce plastic waste and promote sustainable water consumption on campus.

### SDLC Work Overview:
1. **Planning:** Identified user needs and mapped out User Stories in Trello.
2. **Design:** Created Use Case and Class Diagrams to structure the app logic.
3. **Build:** Developed a high-fidelity prototype focusing on map navigation and admin reporting.
4. **Testing:** Conducted user feedback sessions to refine the interface.
