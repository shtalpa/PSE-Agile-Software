# PSE-Agile-Software

### Project title: Where's My Water?
### SDG Goal: 
SDG 6 -- Software can monitor water quality and manage community resources.

### Project Description: 
This application helps university students locate the nearest clean water stations and allows them to report maintenance issues. By ensuring water stations are functional, we reduce plastic waste and promote sustainable water consumption on campus.

### SDLC Work Overview:
1. **Planning:** Identified user needs and mapped out User Stories in Trello.
2. **Design:** Created Use Case and Class Diagrams to structure the app logic.
3. **Build:** Developed a high-fidelity prototype focusing on map navigation and admin reporting.
4. **Testing:** Conducted user feedback sessions to refine the interface.

### Figma Prototype Walkthrough

1. Accessing the Prototype
Open the prototype link.
Click the "Present" (Play icon) in the top right corner.
Tip: If the screen is too zoomed in, press 'Z' on your keyboard twice to fit the prototype to your screen.

2. User Preview
Home Page: Upon landing on the Home Screen, you will see a navigation bar with 5 different interactable buttons: Home (the page you are currently in), Campus Map, Stations, Report, Admin. This page is filled with detailed information on why we established this website that aligns with the SDG 6 values.
-> Quick Links: Easily navigate to respective pages via Quick Link.
Campus Map: You will see a 6-level interactive map of the campus that tells you the status and location of each water station on each floor.
-> Switch Floors: Use the floor-level navigation (Level 1–6) to find water stations on different storeys.
-> Press the "All Water Stations" button to navigate to the Stations page, or "Report an Issue" button to navigate to the Report page.
-> Check Status: Click on any Water Station Icon. It will navigate you to the Stations page for more details.
Stations: You will see a comprehensive list of all water stations.
-> Click on either "Active", "Maintenance", or "Offline" button to filter out the water station list.
-> Click on any Water Station Icon to expand the icon to present more in-depth information of the station. Including a description, features, filter life progress bar, and last checked date.
-> Press the "Report an Issue" button to navigate to the Report page.
Report: When encountering a maintenance issue, you can fill up the report form on this page.
-> Select the issue type (e.g., "Water Qaulity" or "Vandalism/Damaged").
-> Click "Submit Report." You will be redirected to a Success Screen confirming that the Facilities Team has been notified.

3. The Admin Preview
Admin Login: Click the Admin Icon in the navigation bar. Use the credentials (simulated) to enter the dashboard.
-> To logout, click the "Logout" button in the navigation bar to return to Admin Login.
Dashboard (Home): View the System Status, Monthly Sustainable Goals and Urgent Report all in a glance.
-> Click on the "Download Monthly Report" button to download the current monthly report.
-> Click on the "Station Inventory" button to view the list of Stations available on the campus.
-> Click on the "Manage Updates" button to view the active updates of the system.
Register New Station: Allows the admin to register a new water station on the campus by filling up the form with details such as Staion Level, Station Block, Location, Description, and Features. Click on the "Register Water Station" button to register a new station.
User Management: Allows the admin to view Admin List, User Profile, System Control, Audit Log, and Pending Request.
